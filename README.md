## üì¶ Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/devlizer.git
   cd devlizer
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.


## üåê Internationalization

The application supports multiple languages through next-intl:

- **English** (en) - Default language

### Adding New Languages

1. Create a new translation file in the `locales/` directory
2. Update the `i18n` folder files to include the new locale
3. Add the locale to your routing configuration

## üé® Customization

### Styling

- Modify `tailwind.config.ts` for custom design tokens
- Update global styles in `src/app/globals.css`
- Customize component styles in respective component files

### Content

- Update translations in `locales/` directory
- Modify page content in `src/app/[locale]/page.tsx`
- Add new sections in `src/components/sections/`

### Animations

- Customize animations in component files using Framer Motion
- Adjust motion variants for different effects
- Configure scroll-triggered animations
